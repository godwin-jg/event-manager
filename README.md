# Event Manager

A simple event management application built with Next.js, Tailwind CSS, Zustand, and React Hook Form.

## How to Run

```bash
pnpm install
pnpm dev
```

## Path of the Page

- **Home**: `/` - Landing page
- **Events**: `/events` - Event management page with add, search, and delete functionality

## Notes/Assumptions

- **App runs from the Next-Forge Turbo template**
- **State Management**: Zustand with localStorage persistence
- **Forms**: React Hook Form 
- **Project Cleanup**: Cleaned up unnecessary environment variables (Basehub, Sentry, Resend, etc.) from setup
